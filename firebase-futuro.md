# Proxima etapa: Firebase em tempo real

Para transformar o ranking em uma ferramenta ao vivo entre varios computadores, usar:

- GitHub Pages para hospedar o painel.
- Firebase Authentication para login.
- Firebase Firestore ou Realtime Database para sincronizar os dados.

## Perfis sugeridos

### Administrador

- Edita os dados de produtividade.
- Alimenta as horas.
- Cadastra gestores.
- Limpa dados.
- Gera PDF.

### Leitor

- Visualiza rankings.
- Visualiza gestao por hora.
- Visualiza performance por gestor.
- Pode gerar PDF, se liberado.

## Dados que devem ir para o banco

- Nomes.
- Tempos efetivos.
- Unidades.
- Gestores cadastrados.
- Lancamentos hora a hora.
- Configuracoes de metas e cores.
